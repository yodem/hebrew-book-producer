# Hebrew Logical Connectives — Reference Table

The five logical relations and their Hebrew connectors, ordered roughly from most-formal/literary (top) to most-conversational (bottom) within each cell.

| Relation (קשר לוגי) | Connector (מילת קישור) | Use it for | Register |
|---|---|---|---|
| **Addition** (הוספה) | זאת ועוד | Reinforcing a claim with a stronger second example. | Formal-literary |
|  | יתרה מזאת | Adding a stronger restatement. | Formal-literary |
|  | יתר על כן | Same as יתרה מזאת; very formal. | Formal-academic |
|  | נוסף על כך | Adding a parallel point. | Modern-literary |
|  | בנוסף לכך | Same; slightly more conversational. | Modern |
|  | וכן | Listing equivalent items. | Neutral |
|  | גם | Light addition. | Neutral / conversational |
| **Contrast** (ניגוד) | ברם | Strong literary "however". | Formal-literary |
|  | אולם | Standard literary "however". | Formal |
|  | מאידך גיסא | "On the other hand." Strong formal contrast. | Formal-academic |
|  | מנגד | "On the contrary / by contrast." | Formal |
|  | ואילו | "Whereas." Sets up parallel-but-opposing. | Neutral-formal |
|  | בעוד ש… | "While." Same as ואילו but introduces subordinate clause. | Neutral |
|  | אבל | Conversational "but." | Conversational |
|  | אך | Slightly less conversational than אבל. | Neutral |
| **Cause** (סיבה) | שכן | "For / because." Literary. | Formal-literary |
|  | מחמת ה… | "Due to the…" | Formal |
|  | בגין | "Owing to." | Formal-legal |
|  | הואיל ו… | "Inasmuch as." Strong literary. | Formal-literary |
|  | לאור | "In light of." | Neutral-formal |
|  | מכיוון ש… | "Because." | Neutral |
|  | משום ש… | "Because." Slightly more literary than מכיוון ש. | Neutral |
|  | כי | Conversational "because." | Conversational |
| **Result** (תוצאה) | לפיכך | "Therefore." Standard literary. | Formal |
|  | אי לכך | "In consequence." | Formal-academic |
|  | מכאן נובע ש… | "It follows that." | Formal |
|  | עקב כך | "As a result." | Neutral-formal |
|  | על כן | "Therefore." | Neutral-formal |
|  | לכן | "Therefore." Most common. | Neutral |
|  | ולכן | Same with conjunction. | Neutral |
|  | אז | Conversational "so." | Conversational |
| **Concession** (ויתור) | חרף | "Despite." Strong literary. | Formal-literary |
|  | על אף | "In spite of." | Formal |
|  | אף על פי ש… | "Even though." | Formal |
|  | עם כל זאת | "Nevertheless." | Neutral-formal |
|  | למרות ש… | "Despite the fact that." | Neutral |
|  | בכל זאת | "All the same." | Neutral |
|  | אף ש… | "Although." | Neutral-formal |

## Rules of use

1. **Match the register of the surrounding text.** A formal-literary chapter should not switch to "אבל" mid-paragraph.
2. **Do not double-load relations.** "אולם, נוסף על כך…" combines Contrast + Addition in a way that confuses the reader. Pick one.
3. **Frequency target:** in one printing sheet (24,000 chars), expect 4–8 instances of each relation. A run of zero of one relation across a whole chapter is a structural defect — usually meaning the argument is one-sided.
4. **For philosophy / academic:** prefer the formal-literary register. "ברם", "שכן", "אי לכך", "מאידך גיסא" mark this register strongly.
5. **For autobiography:** mix registers deliberately. The literary connectors signal reflection; the conversational ones signal scene.
6. **For religious texts:** use the formal-literary register and prefer the connectors with deeper roots in classical Hebrew ("הואיל ו…", "אי לכך", "ברם"); avoid conversational ones.

## Banned mixings

| Avoid | Why |
|---|---|
| "אולם, מאידך גיסא" | Two contrast connectors in one breath. |
| "ולפיכך, מנגד" | Result + Contrast. Choose one. |
| "שכן, אבל" | Cause + Contrast back-to-back. Restructure. |
| Three connectors of any kind in one sentence | Almost certainly a sign of muddled logic. Restructure. |
